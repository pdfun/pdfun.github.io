iOS简单实用category

常用第三方category,持续更新中...

AutoCoding

AutoCoding is a category on NSObject that provides automatic support for NSCoding to any object.

BlocksKit

Blocks in C and Objective-C are downright magical. They make coding easier and potentially quicker, not to mention faster on the front end with multithreading and Grand Central Dispatch. BlocksKit hopes to facilitate this kind of programming by removing some of the annoying - and, in some cases, impeding - limits on coding with blocks.

UIView

UIView-Positioning

UIView+Positioning is an Objective-C Category which provides easy shorthand methods to defining the frame properties (width, height, x, y) of any UIView based object in an easy fashion, as well as extra helpful properties and methods

UIView-i7Rotate360

This UIView Categorie (Extend) will allow you to rotate a view easly 360°

UIImageView-PlayGIF

UIImageView category/subclass for playing GIF. Simple, high performance, low memory footprint.

UITextField-Shake

UITextField category that adds a shake animation like the password field of the OsX login screen.
