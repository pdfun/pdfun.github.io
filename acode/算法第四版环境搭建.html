《算法》第四版Mac环境手动搭建

算法第四版 macOS High Sierra 手动搭建运行环境

安装java

brew cask install java

配置java环境变量

vi ~/.bashrc export JAVA_HOME="/Library/Java/JavaVirtualMachines/jdk-10.0.2.jdk/Contents/Home" CLASSPATH="$JAVA_HOME/lib"

导入jar包

mkdir ~/algs4 到配套网站下载algs4 jar包，并拷贝到 ~/algs4 目录下 vi ~/.bashrc export CLASSPATH=$CLASSPATH:~/algs4/algs4.jar source ~/.bashrc

运行测试

  • 解压下载的jar包到xx目录,直入xx目录
  • javac TestAlgs4.java
  • java TestAlgs4
  • 看到下面的界面就代表成功

参考网站

!https://algs4.cs.princeton.edu/code/,滚动到下面 Installing the textbook libraries.有配置介绍。
